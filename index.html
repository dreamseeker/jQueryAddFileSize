jQuery AddFileSize プラグイン

ライセンス：MITライセンス
ダウンロード：GitHub
バージョン：1.00

これは、リンク先に指定されたファイルの容量を取得して、a要素の直後にファイルサイズを表示するためのjQuery プラグインです。同一ドメインで、URLまたは絶対パスで指定されたファイルが対象となります。

なお、ファイルサイズがKBの場合には小数点1位、MBまたはGBの場合は小数点2位までの表示となります。

更新履歴

2012.02.04
jQuery AddFileSize プラグイン ver 1.00を公開。

サンプル1

  • PDFファイルのサンプル
  • Wordファイルのサンプル
$('a[href$=".pdf"], a[href*=".doc"]').AddFileSize({
	php_url: '/path_to/filesize.php'
});

/lib/filesize.phpと/js/jquery.AddFileSize.jsをアップロード後、プラグイン本体をscript要素で読み込む必要があります。

サンプル2

  • PDFファイルのサンプル
  • Wordファイルのサンプル
$('a[href$=".pdf"], a[href*=".doc"]').AddFileSize({
	php_url: '/path_to/filesize.php',
	extension: {
		enable: true
	}
});

サンプル3

  • PDFファイルのサンプル
  • Wordファイルのサンプル
$('a[href$=".pdf"], a[href*=".doc"]').AddFileSize({
	php_url: '/path_to/filesize.php',
	prefix: '【',
	suffix: '】',
	extension: {
		enable: true,
		suffix: '形式・'
	}
});

初期化オプションについて（グレーの枠内は初期値です）

php_url:
ファイルサイズを取得するためのPHPファイル（/lib/filesize.php）へのURLもしくは絶対パスを指定します。
php_url: '/lib/filesize.php'
prefix:
ファイルサイズ表記の接頭辞を指定します。
prefix: ' ( '
suffix:
ファイルサイズ表記の接尾辞を指定します。
suffix: ' )'
extension:
enable:
ファイル拡張子の表示をブーリアン（ true | false ）で指定します。
doc（または、docx）が「Word」、xls（または、xlsx）が「Excel」、ppt（または、pptx）が「PowerPoint」に置き換えられます。それ以外であれば、半角英字（大文字）表記となります。
suffix:
ファイル拡張子を表示する際の接尾辞を指定します。
extension: {
	enable: false,
	suffix: ' / '
}
